# LHL-Mobile
LHL mobile App


# Prerequisites:

Node 12~

Ionic 4.12~

Angular 10~

Android sdk and gradle required for android build

Xcode and cocoapods required for ios build


# Build Steps:


cd to lhl-mobile/ble

run npm i

then ionic cordova platform add android

then ionic cordova build android
